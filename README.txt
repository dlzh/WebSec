There are multiple description files in this project:

1. .github/workflows/maven.yml 
Tells the Github action how to compile and pakcage a maven file and related information.
Or maybe other name such as ci.yml -- it's your choice.
2. buildspec.yml 
This is a file that is used by AWS CodeBuild. 
3. Dockerfile 
This is a configuration file that is used to create a Docker image.
